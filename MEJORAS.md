## 🎨 Paleta de Efectos

### Sombras
- `shadow-sm`: Sutil para elementos elevados
- `shadow-md`: Media para cards y contenedores
- `shadow-lg`: Grande para modales y destacados
- `shadow-glow`: Resplandor para efectos especiales

### Transiciones
- `transition-fast`: 0.2s para cambios rápidos
- `transition-smooth`: 0.4s cubic-bezier suave
- `transition-bounce`: 0.5s con rebote elegante

### Animaciones
- `fadeIn`: Entrada suave de página
- `float`: Movimiento vertical sutil
- `pulse-glow`: Resplandor pulsante
- `pulse-ring`: Anillo expansivo
