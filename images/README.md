# Carpeta de Imágenes

## Imágenes Requeridas

Coloque las siguientes imágenes en esta carpeta:

### Obligatorias:
- `dr-checa.jpg` - Fotografía profesional del Dr. David Checa (recomendado: 800x800px)

### Opcionales:
- `logo.png` - Logo del consultorio (si existe)
- `cert-placeholder.png` - Logos de certificaciones e instituciones
- Imágenes para artículos del blog

## Especificaciones Técnicas

### Formatos recomendados:
- JPG para fotografías
- PNG para logos y gráficos con transparencia
- WebP para mejor compresión (opcional)

### Tamaños recomendados:
- Fotografía del doctor: 800x800px
- Logos: 200x200px
- Imágenes de blog: 1200x630px

### Optimización:
- Comprimir todas las imágenes antes de subirlas
- Usar herramientas como TinyPNG o Squoosh
- Mantener el tamaño de archivo bajo 500KB cuando sea posible

## Nota Importante

Mientras no se proporcionen las imágenes reales, el sitio funcionará correctamente pero mostrará imágenes rotas en algunas secciones. Esto no afecta la funcionalidad del sitio.
